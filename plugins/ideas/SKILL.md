# ideas

Capture and review ideas organized by year

## Commands
- `/idea` (WRITE)
- `/ideas`

## Intent
USE FOR: Capture startup ideas, project ideas, or brainstorms for later review

## Primitives
```
/idea <description> — Capture an idea
/ideas — Show this year's ideas
```

## Rules
- Do NOT invent subcommands — only use commands listed above
- For write commands, use the EXACT flag syntax from Primitives above
- Respect explicit timezones in user input
