# image_gen

Generate images using Gemini

## Commands
- `/img`

## Intent
USE FOR: Generate images from text descriptions

## Primitives
```
/img <prompt> — Generate an image (flash model)
/img pro <prompt> — Generate with higher quality (pro model)
```

## Rules
- Do NOT invent subcommands — only use commands listed above
- Respect explicit timezones in user input
